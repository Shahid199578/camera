# Intex - Enterprise Security & Surveillance

This is the Official React frontend for the Intex Security & Surveillance brand.

## Quick Start

This project is built using [React](https://react.dev/), [Vite](https://vitejs.dev/), [TypeScript](https://www.typescriptlang.org/), and [Tailwind CSS](https://tailwindcss.com/).

To get the project running locally, follow these steps:

### 1. Install Dependencies
Run the following command in the root directory to install all required Node modules:
```bash
npm install
```

### 2. Run the Development Server
Start the local development server:
```bash
npm run dev
```

The application will typically be accessible at `http://localhost:5173`. Open this URL in your browser to view the site.

### 3. Build for Production
To create a production-ready build:
```bash
npm run build
```
This will compile and minify the assets into the `dist` folder.
